## git messages should start with 
 
`feat: <this is a feature>`
`fix: <this is a fix>`
`build: <for builds>`
`chore: <chores>`
`ci: <for ci related tasks`
`docs: <documentation>`
`style: <changing style>`
`refactor: <code refactoring commits>`
`test: <code testing>`
  
  
## message body

```
Resolves #10
With: Jonh`
```
